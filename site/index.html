Sprint Bot – GitHub Pages Demo

This page hosts the floating widget and loads the UI in an iframe.
How it works
  • This page includes widget.js from the same folder.
  • window.SprintBot.config.origin points to your UI site.
  • We append ?api=... so the UI talks to your API.
Tip
  • Set --sb-accent to change button color.
  • Ensure your API is HTTPS when embedding on GitHub Pages.
